# PiPhone Agents

> Родина кишенькових AI-агентів на Android. Один агентний цикл — різні ролі.

## Філософія

Ми не будуємо монолітного «Джарвіса», який намагається робити все.
Ми будуємо **родину спеціалізованих агентів** на спільному ядрі:

1. **Координатор** — виконання команд через Android Intents (текст/голос)
2. **Секретар** — голосовий асистент через Bluetooth-гарнітуру (hands-free)
3. **Проактивний** — фоновий помічник, що сам пропонує дії

## Чому не один агент на всі випадки?

Кожна роль має різні вимоги:

| Агент | Тригер | Інтерфейс | Батарея | Інтернет |
|---|---|---|---|---|
| Координатор | Ти кажеш | Екран + TTS | Низько | Опціонально |
| Секретар | Ти кажеш | Голос ↔ TTS | Середньо | Потрібен |
| Проактивний | Час/подія | Сповіщення | Фон (мало) | Опціонально |

Спільне ядро гарантує, що всі три сумісні, розширювані та зворотно сумісні.

## Репозиторій

```
piphone-agents/
├── core/                    # Спільне ядро (Python)
│   ├── agent_loop.py        # Агентний цикл (plan → execute → check)
│   ├── action_manager.py    # Реєстр дій/інтентів
│   ├── tool_registry.py     # Реєстр інструментів
│   ├── plugin_api.py        # API для плагінів/гостей
│   ├── llm_client.py        # HTTP-клієнт для LLM
│   ├── cactus_adapter.py    # Cactus Needle інтеграція
│   └── android_intents.py   # База Android Intent схем
├── agents/
│   ├── coordinator/         # APK координатора
│   ├── assistant-secretary/ # APK секретаря
│   └── assistant-proactive/ # APK проактивного
├── guests/                  # Плагіни-гості
│   ├── guest_base.py        # Базовий клас гостя
│   ├── guest_messaging.py   # WhatsApp, Telegram, SMS
│   ├── guest_media.py       # YouTube, Music
│   ├── guest_navigation.py  # Maps, GPS
│   ├── guest_notes.py       # Нотатки, нагадування
│   └── guest_system.py      # Системні дії
├── docs/                    # Документація
│   ├── 01_architecture.md
│   ├── 02_guest_api.md
│   └── 03_plans.md
└── README.md
```
